# Фразодром / Frazodrom

Открытый бесплатный тренажёр английского для русскоязычных: переводишь предложения с русского на английский по грамматическим паттернам (A1–C1), доводишь конструкции до автоматизма. Интервальные повторения (FSRS), аудирование и говорение, экзамены, ИИ-репетитор по твоему собственному API-ключу. Работает в браузере, без регистрации, весь прогресс хранится локально.

Free and open-source English pattern trainer for Russian speakers: translate sentences RU→EN, drill grammar patterns to automaticity. Spaced repetition (FSRS), listening/speaking, exams, BYOK AI tutor. Runs in the browser, no accounts, local-first.
